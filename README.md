# 🚀 AI Agents Team - نظام الأرباح الذكي

> فريق من 9 وكلاء اصطناعيين يعملون بشكل تلقائي لجني الأرباح من الإنترنت

## 🤖 عن المشروع

هذا المشروع هو نظام متكامل يديره **9 وكلاء ذكيين** (AI Agents) يعملون بشكل تلقائي بالكامل لجني أرباح حقيقية من منصات متعددة. المشرف العام (أنت) فقط تقوم بالإشراف واتخاذ القرارات الاستراتيجية.

### 🎯 الهدف
- تحقيق أرباح حقيقية من الإنترنت بشكل تلقائي
- تقديم خدمات تقنية عن بعد
- تطوير وبيع ألعاب الفيديو (مستقبلاً)

---

## 🧠 هيكل الفريق (9 وكلاء)

| الوكيل | المسؤولية |
|--------|-----------|
| **المدير التنفيذي** | إدارة الفريق واتخاذ القرارات التشغيلية |
| **محلل البيانات** | تحليل الأداء والأرباح وتقديم التقارير |
| **مدير المنصات** | إدارة حسابات المنصات (urpay, Amazon, Upwork...) |
| **مدير الخدمات** | تنفيذ الخدمات التقنية عن بعد |
| **المسوق الرقمي** | التسويق وجلب العملاء |
| **مدير المحتوى** | إنشاء المحتوى للتسويق |
| **المحاسب** | تتبع الأرباح والمصروفات |
| **مطور الويب** | تطوير وصيانة الواجهات والـ API |
| **مطور الألعاب** | تطوير الألعاب (مستقبلاً) |

## 🛠️ التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|-----------|
| **Node.js + Express** | تشغيل السيرفر الخلفي |
| **Supabase** | قاعدة البيانات (تخزين الأرباح) |
| **GitHub Pages** | استضافة الموقع |
| **GitHub Actions** | التشغيل التلقائي كل ساعة |
| **REST API** | واجهة التواصل مع الوكلاء |

---

## 📡 الـ API Endpoints

| الطريقة | المسار | الوصف |
|---------|--------|-------|
| `GET` | `/api/earnings` | جلب جميع الأرباح |
| `GET` | `/api/earnings/total` | جلب إجمالي الأرباح |
| `POST` | `/api/earnings` | إضافة ربح جديد |
| `PUT` | `/api/earnings/:id` | تحديث حالة ربح |
| `GET` | `/api/reports/platform` | تقرير حسب المنصة |

### مثال إضافة ربح جديد:
```bash
curl -X POST https://[username].github.io/[repo]/api/earnings \
  -H "Content-Type: application/json" \
  -d '{
    "source": "urpay",
    "amount": 50.00,
    "platform": "urpay",
    "status": "completed"
  }'
